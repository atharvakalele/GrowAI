## ⚡ Quick Rule Recall (Top 5 Critical Rules)

1. **RESEARCH FIRST** — Always check official docs before coding. Never assume old methods work.
2. **API VERSION CHECK** — Verify which API version we use. Only write compatible code.
3. **TEST BEFORE DELIVER** — Test every feature yourself before giving final output.
4. **360° IMPACT CHECK** — Before any change, check if it breaks existing features.
5. **NO HARDCODING** — Use variables/config for all important values.

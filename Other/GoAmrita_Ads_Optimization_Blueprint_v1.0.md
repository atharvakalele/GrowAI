# PART 2: 🎯 AD OPTIMIZATION STRATEGY FRAMEWORK

## 2.1 Current Problem Diagnosis

### What's likely WRONG with your campaigns right now:
1. ❌ **No negative keywords** — irrelevant searches pe paisa waste ho raha hai
2. ❌ **Auto campaigns only** — Amazon ka auto targeting bahut loose hota hai
3. ❌ **Same bids for all keywords** — high-performer aur low-performer pe same paisa
4. ❌ **No search term analysis** — pata hi nahi kaun se words convert ho rahe hain
5. ❌ **No dayparting** — raat 2 baje bhi same bid jab koi kharid nahi raha
6. ❌ **No campaign structure** — sab ek jagah mix hai
7. ❌ **No bid adjustments** — years se same bid, market change ho gaya
8. ❌ **Budget bleeding** — budget subah khatam, shaam ko high-conversion time pe ad nahi dikhta

## 2.2 Campaign Structure Strategy (3-Tier System)

### For each product (or product group), 3 campaigns:

```
🏗️ CAMPAIGN STRUCTURE PER PRODUCT/GROUP:

├── 📦 Campaign 1: AUTO (Research/Discovery)
│   ├── Purpose: New keyword discovery
│   ├── Budget: 20% of total
│   ├── Bid: Low-Medium
│   ├── Targets: Amazon's auto-targeting
│   └── Action: Harvest winning search terms → move to Manual
│
├── 🎯 Campaign 2: MANUAL BROAD/PHRASE (Growth)
│   ├── Purpose: Test keywords at broader match
│   ├── Budget: 30% of total  
│   ├── Bid: Medium
│   ├── Match types: Broad + Phrase
│   └── Action: Winners → move to Exact, Losers → Negative
│
└── 💎 Campaign 3: MANUAL EXACT (Profit)
    ├── Purpose: Maximum profit from proven keywords
    ├── Budget: 50% of total
    ├── Bid: Highest (but controlled)
    ├── Match type: Exact only
    └── Action: Tight bid management, scale winners
```

### Why 3-Tier?
- **AUTO** → Discover what customers actually search
- **BROAD/PHRASE** → Test those discoveries at wider reach
- **EXACT** → Lock in profit from proven terms
- This is the **Search Term Harvesting Funnel** — industry-proven method

## 2.3 Keyword Strategy

### Keyword Harvesting Flow:
```
AUTO Campaign (search term report)
    ↓ (keywords with 2+ orders, ACoS < target)
MANUAL BROAD/PHRASE Campaign
    ↓ (keywords with 5+ orders, good ACoS consistently)
MANUAL EXACT Campaign (profit machine)
    
At every stage:
    → Poor performers (high spend, 0 sales) → NEGATIVE KEYWORD list
```

### Negative Keyword Strategy (MONEY SAVER #1):
- **Immediate negatives:** competitor brand names (unless intentional), irrelevant terms
- **Performance negatives:** search terms with 15+ clicks but 0 sales
- **Category negatives:** terms from wrong product category
- **Review weekly:** add new negatives from search term report

### Bid Optimization Rules:

| Scenario | Action |
|----------|--------|
| ACoS < 10% (very profitable) | 🔼 Increase bid 15-20% — scale this keyword |
| ACoS 10-15% (profitable) | 🔼 Increase bid 10% — grow slowly |
| ACoS 15-25% (target range) | ✅ Maintain bid — sweet spot |
| ACoS 25-35% (warning zone) | 🔽 Decrease bid 10-15% |
| ACoS 35-50% (losing money) | 🔽 Decrease bid 25% or pause |
| ACoS > 50% (bleeding) | 🛑 Pause keyword, add to review list |
| High impressions, 0 clicks | Review ad copy/listing, maybe reduce bid |
| High clicks, 0 sales | 🔽 Add as negative OR fix product listing |
| 0 impressions | 🔼 Increase bid, keyword may be too low |

### Placement Strategy:
- **Top of Search (first page):** Best for proven exact keywords — higher CPC but higher conversion
- **Product Pages:** Good for competitor targeting — lower CPC
- **Rest of Search:** Default, balanced approach

## 2.4 Budget Rules (Dynamic Budget System)

### Daily Budget Allocation:
```
Total Daily Budget for a product = Based on performance

IF product ACoS < 15%  → Increase budget 20% (it's making money!)
IF product ACoS 15-25% → Maintain budget (healthy)
IF product ACoS 25-35% → Reduce budget 15%
IF product ACoS > 35%  → Reduce budget 30% or pause
IF product has 0 sales in 7 days → Pause campaign, fix listing first
```

### Monthly Budget Calendar:
```
1st-5th:   Budget × 1.3 (salary season — HIGH spend)
6th-10th:  Budget × 1.1
11th-20th: Budget × 1.0 (normal)
21st-25th: Budget × 0.8 (save)
26th-31st: Budget × 0.7 (month end)
```

### Module 2: ANALYSIS ENGINE (Core Logic)

#### Analysis 1: Search Term Analysis
```
For each search term in last 7 days:
├── IF orders >= 2 AND ACoS < target → HARVEST (move to manual exact)
├── IF clicks >= 15 AND orders = 0 → NEGATIVE (block this term)
├── IF impressions > 1000 AND clicks = 0 → LOW RELEVANCE (review)
├── IF ACoS < 10% → STAR PERFORMER (increase bid)
└── IF spend > ₹500 AND orders = 0 → IMMEDIATE BLOCK
```

#### Analysis 2: Campaign Health Check
```
For each campaign daily:
├── Budget utilization % (is budget running out too early?)
├── ACoS trend (improving or worsening?)
├── Impression share (are we visible enough?)
├── Click-through rate (is our ad compelling?)
├── Conversion rate (is our listing good enough?)
└── ROAS (Return on Ad Spend)
```

#### Analysis 3: Product-Level Profitability
```
For each product:
├── Total ad spend vs total ad sales
├── TACoS (Total ACoS including organic)
├── Profit margin AFTER ad spend
├── Is this product worth advertising?
└── Category benchmark comparison
```

#### Analysis 4: Keyword Performance Matrix
```
          HIGH Clicks
              │
    OPTIMIZE  │  SCALE
    (fix listing)│ (increase bid+budget)
   ───────────┼───────────
    PAUSE     │  MONITOR
    (waste)   │  (potential)
              │
          LOW Clicks
    LOW Sales ←──────→ HIGH Sales
```

#### Analysis 5: Day/Hour Performance (Dayparting)
```
Track performance by hour:
├── Which hours have best ACoS?
├── Which hours have worst ACoS?
├── Recommend bid multipliers per hour
└── Auto-adjust based on data
```

# PART 5: 📊 DAILY OPTIMIZATION RULES

## Rule 1: Daily Must-Do Checklist
Every day the system (or you) must:
- [ ] Pull fresh data from API
- [ ] Check search term report → add negatives, harvest winners
- [ ] Review bid performance → adjust bids per rules
- [ ] Check budget utilization → redistribute if needed
- [ ] Review any alerts
- [ ] Log all changes in changelog
- [ ] Compare today vs yesterday → spot anomalies

## Rule 2: Weekly Must-Do
Every week:
- [ ] Aggregate weekly performance
- [ ] Compare week-over-week trends
- [ ] Review all negative keywords added → any wrongly blocked?
- [ ] Check campaign structure → any campaign dominating budget?
- [ ] Review product-level performance
- [ ] Update strategy based on weekly data

## Rule 3: Monthly Must-Do
Every month:
- [ ] Full monthly performance review
- [ ] Strategy effectiveness analysis (what worked, what didn't)
- [ ] Budget planning for next month
- [ ] Festival/event preparation
- [ ] Clean up dormant campaigns
- [ ] Review and update optimization rules

## Rule 4: Change Monitoring Protocol
For EVERY change made:
```
BEFORE making change:
├── Record: current state (bid, budget, status)
├── Record: reason for change
└── Record: expected impact

AFTER change (next day):
├── Record: actual impact
├── Compare: expected vs actual
└── Learn: was this a good or bad decision?
```

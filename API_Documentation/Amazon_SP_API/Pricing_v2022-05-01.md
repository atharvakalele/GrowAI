# Pricing API v2022-05-01 — Official Reference
**Source:** Amazon GitHub OpenAPI Spec
**Our Endpoint:** sellingpartnerapi-eu.amazon.com

## Endpoints (2 batch operations only)

### 1. Get Featured Offer Expected Price (Batch)
- **Path:** POST /batches/products/pricing/2022-05-01/offer/featuredOfferExpectedPrice
- **Rate:** 0.033 req/sec (burst: 1) — VERY SLOW, plan accordingly!
- **Batch Size:** Up to 40 items per request

**Purpose:** Returns the price at or below which a seller can expect to WIN the Buy Box (Featured Offer).

**Request Body:**
```json
{
  "requests": [{
    "uri": "/products/pricing/2022-05-01/offer/featuredOfferExpectedPrice",
    "method": "GET",
    "marketplaceId": "A21TJRUUN4KGV",
    "sku": "your-sku-here"
  }]
}
```

**Response Fields:**
- featuredOfferExpectedPrice.listingPrice: { currencyCode, amount } — TARGET PRICE to win Buy Box
- resultStatus: VALID_FOEP | NO_COMPETING_OFFERS | OFFER_NOT_ELIGIBLE | OFFER_NOT_FOUND | ASIN_NOT_ELIGIBLE
- competingFeaturedOffer: current Buy Box winner details
- currentFeaturedOffer: your current offer details

### 2. Get Competitive Summary (Batch)
- **Path:** POST /batches/products/pricing/2022-05-01/items/competitiveSummary
- **Rate:** 0.033 req/sec (burst: 1)
- **Batch Size:** Up to 20 items

**Purpose:** Returns competitor pricing, Buy Box info, reference prices.

**Request Body:**
```json
{
  "requests": [{
    "asin": "B0FBWXQ9JL",
    "marketplaceId": "A21TJRUUN4KGV",
    "includedData": ["featuredBuyingOptions", "referencePrices", "lowestPricedOffers"],
    "method": "GET",
    "uri": "/products/pricing/2022-05-01/items/competitiveSummary"
  }]
}
```

**includedData options:** featuredBuyingOptions, referencePrices, lowestPricedOffers, similarItems

**Response — featuredBuyingOptions:**
- buyingOptionType: "New"
- segmentedFeaturedOffers:
  - sellerId, condition, subCondition
  - fulfillmentType: AFN (FBA) | MFN (self-ship)
  - listingPrice: { currencyCode, amount } — CURRENT Buy Box price
  - shippingOptions: [{ shippingOptionType, price }]
  - primeDetails: { eligibility: NATIONAL | REGIONAL | NONE }
  - featuredOfferSegments: [{ customerMembership: PRIME|NON_PRIME|DEFAULT }]

**Response — referencePrices:**
- name: string (e.g., "WAS_PRICE", "LIST_PRICE")
- price: { currencyCode, amount }

**Response — lowestPricedOffers:**
- offers: [{ sellerId, condition, fulfillmentType, listingPrice, shippingOptions }]

## Key Capabilities for Our System:
- **Buy Box Price:** featuredBuyingOptions → listingPrice
- **Buy Box Winner:** featuredBuyingOptions → sellerId (is it us?)
- **Competitor Prices:** lowestPricedOffers → all competing prices
- **Target Price to Win:** getFeaturedOfferExpectedPrice → listingPrice
- **Reference/MRP:** referencePrices

## CRITICAL Limitations:
- Only 0.033 req/sec = ~2 requests per MINUTE!
- 300 products ÷ 40 per batch = 8 batches ÷ 0.033 = ~4 minutes minimum
- MUST implement rate limiter + queue
- getFeaturedOfferExpectedPrice requires SKU (not ASIN) — need SKU mapping

## Error Codes: 400, 401, 403, 404, 429, 500, 503

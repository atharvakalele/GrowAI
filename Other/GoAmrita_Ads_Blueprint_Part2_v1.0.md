## 2C.2 Key AI Techniques Used by Top Tools

### 1. Predictive Bidding
- Algorithm analyzes: time of day, day of week, keyword competition, conversion history
- Predicts: "this keyword will convert at X% in the next hour"
- Sets bid: to maximize profit at predicted conversion rate
- **Our approach:** Rule-based first, learning layer on top

### 2. Search Term Scoring
- Each search term gets a score based on: CTR, conversion rate, ACoS, relevance
- High score → harvest to manual campaign
- Low score → add as negative
- **Our approach:** IQ scoring system (0-100)

### 3. Budget Pacing
- Problem: budget runs out at 11 AM, misses 6-10 PM peak
- Solution: pace budget hourly — spend less in morning, save for evening
- Amazon now supports "Bid Schedule Rules" (increase only, not decrease)
- **Our approach:** Campaign-level budget rules + hourly awareness

### 4. Anomaly Detection
- Sudden ACoS spike? CPC jump? Impressions drop?
- AI detects these automatically
- **Our approach:** Day-over-day comparison with threshold alerts

```
📧 MULTI-CHANNEL ALERT SYSTEM:

TYPE 1: DAILY REPORT EMAIL
├── Time: Configurable (e.g., 8 AM IST)
├── Content: Summary of yesterday's performance
├── Attachment: Full daily Excel report
├── Auto-open: Mac script opens Excel automatically
└── Frequency: Daily

TYPE 2: CRITICAL ALERT EMAIL
├── Time: Immediately when detected
├── Triggers: Budget exhausted, ACoS > 60%, API error
├── Content: What happened + What to do
├── No attachment (speed matters)
└── Frequency: As needed

TYPE 3: WEEKLY SUMMARY EMAIL
├── Time: Every Monday 9 AM
├── Content: Full week review
├── Attachment: Weekly summary Excel
├── Strategy effectiveness report included
└── Frequency: Weekly

TYPE 4: MONTHLY REVIEW EMAIL  
├── Time: 1st of every month
├── Content: Full month analysis
├── Attachment: Monthly Excel + strategy report
├── Budget recommendation for next month
└── Frequency: Monthly

MULTI-EMAIL SUPPORT:
├── Different alert types → can go to different emails
├── Config: { "daily": ["msir@email.com"], 
│             "critical": ["msir@email.com", "team@email.com"],
│             "weekly": ["msir@email.com"] }
└── Fully configurable in JSON

MAC AUTO-OPEN FEATURE:
├── Mac launchd job / cron
├── At configured time → opens latest report Excel
├── User finds report already open on laptop
├── Zero effort to start daily review
└── Script: open -a "Microsoft Excel" latest_report.xlsx
```
